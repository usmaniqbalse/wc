== Description ==

This plugin will be used to display more than one author name on a post.

== Installation ==

1. Upload `wc-wordpress-contributors.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How you can select contributor for the blog post? =

Go to the post > edit post - you'll see contributors metabox to select contributor for the post.
